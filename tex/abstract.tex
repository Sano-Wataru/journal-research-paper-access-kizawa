% 提案システムの目的
We propose an edge computing system that estimates the activity levels of office workers using a light-detection-and-ranging (LiDAR) sensor network.
% 点群の特徴抽出
The system captures point-cloud data of individuals at work obtained by LiDAR, converting it into numerical features for analysis.
% 機械学習による推論
Machine learning models are employed to estimate the activity levels from the features.
% 評価実験
To evaluate the system, we conducted experiments in experimental office environments, collecting point-cloud data while participants watched educational videos.
% 活動レベルの評価
Activity indicators were derived from tests and questionnaires, and binary labels were generated using clustering for model evaluation.
% モデルの評価
We developed several models utilizing different data types and feature extraction techniques, assessing their performance against the binary labels.
% 結果
Our results demonstrated the effectiveness of the proposed system for estimating activity levels based on a range of three-dimensional features.
